# Setup and Run

Use this file if you want quick instructor-facing instructions.

Local development:

```bash
npm install
npm run dev
```

Build and preview:

```bash
npm run build
npm run preview
```

Codespaces / Devcontainer:

- Open the repo in Codespaces or use Remote - Containers. The container will run `npm install` on create.
- Run `npm run dev` in the Codespaces terminal.

Ports:

- Vite dev server: 5173 (forwarded)
- Preview: 5174
